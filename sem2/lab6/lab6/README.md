## Цель работы

Разработать мобильное приложение на Android (Kotlin) для взаимодействия с Spring Boot приложением фитнес-клуба, обеспечивающее управление участниками и тренировками через мобильные устройства.

## Выполнение работы

Создан API клиент с Retrofit:

Настроен базовый URL для эмулятора (http://10.0.2.2:8080/)

Реализованы все необходимые endpoints

Добавлен логгинг запросов для отладки

Разработан пользовательский интерфейс:

MainActivity - главный экран с навигацией

MembersActivity - список участников

MemberDetailActivity - детальная информация об участнике

WorkoutsActivity - список тренировок

Адаптивная верстка с ConstraintLayout

Реализована бизнес-логика:

Асинхронные запросы через Kotlin Coroutines

Обработка ошибок сети

Кэширование данных при необходимости

Обновление UI в главном потоке

Добавлены дополнительные функции:

Pull-to-Refresh для обновления списков

Поиск и фильтрация участников

Валидация данных при добавлении

Поддержка темного режима

Настроена сборка и развертывание:

Конфигурация Gradle для разных сборок

Минимальная версия API 24 (Android 7.0)

Разрешения для доступа в интернет
