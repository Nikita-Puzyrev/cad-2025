## Цель работы

Разработать мобильное приложение для управления магазином.

## Выполнение работы

Создано Android приложение на Kotlin. Использованы Retrofit для API, RecyclerView для списков, Material Design для интерфейса. Реализованы: список товаров, детальный просмотр, добавление/редактирование через диалоги. Настроено подключение к localhost через эмулятор.
## Выводы

Разработано нативное мобильное приложение с полным CRUD функционалом. Демонстрирует архитектуру MVVM и работу с REST API. Готово к расширению и публикации.
