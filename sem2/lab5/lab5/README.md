## Цель работы

Создать веб-интерфейс для работы с API музыкального магазина.

## Выполнение работы

Разработан одностраничный веб-клиент на HTML/CSS/JS. Реализованы: отображение товаров в виде карточек, поиск, CRUD операции через модальные окна, проверка подключения к серверу. Использован Fetch API для работы с REST. Клиент работает в браузере без установки.

## Выводы

Создан легкий и функциональный веб-клиент. Демонстрирует возможность создания SPA без фреймворков. Готов к использованию с запущенным Spring Boot сервером.
