## Цель работы

Разработать серверную часть приложения "Музыкальный магазин" на Spring Boot с REST API для управления товарами.

## Выполнение работы
Создано Spring Boot приложение с Spring Data JPA и SQLite. Реализованы модели Product и Order, репозитории JPA, REST контроллеры с CRUD операциями. Настроена база данных SQLite с кастомным диалектом. Добавлена конфигурация CORS для клиентов. Приложение запускается на порту 8080.


## Выводы

Создан работоспособный backend с полным REST API. Решены проблемы совместимости Spring Boot и SQLite. Приложение готово к подключению клиентов.
